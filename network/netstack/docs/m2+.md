# M2+：TCP 扩展（四元组 demux、Connect、Listener backlog）

在 [M2 基础验收](m2.md) 之上，本阶段补齐 todo 中推迟的两项：

| 项 | 说明 |
|----|------|
| **Demuxer 四元组** | `ESTABLISHED` / 半连接按完整四元组查找 |
| **Connect** | 客户端主动打开（SYN-SENT → ESTABLISHED） |
| **Listener + backlog** | 监听套接字常驻 LISTEN；SYN 时 fork `Connection` |

仍 **不包含**：RTO/重传、拥塞控制、FIN-WAIT/TIME-WAIT 完整路径、TUN。

---

## 架构变化

```text
                    listen_endpoints_
                    (local_ip, local_port)
                           │
                           ▼
                    tcp::Listener  ──SYN──►  new tcp::Connection
                           │                    │
                           │                    │ RegisterConnectedEndpoint
                           │                    ▼
                           │             connected_endpoints_
                           │                 (四元组)
                           │                    │
                           └──── Accept() ◄─────┘ ESTABLISHED
```

| 类型 | 类 | demuxer 表 | RFC 793 状态 |
|------|-----|------------|--------------|
| 监听 | `tcp::Listener` | 监听键 | LISTEN（常驻） |
| 连接 | `tcp::Connection` | 四元组 | SYN-SENT … CLOSED |

原 `tcp::Endpoint` 已拆分为 `Listener` + `Connection`。

---

## API 速览

### `tcp::Listener`

```cpp
Listener ln(&stack);
ln.Listen(nic_id, {10,0,0,1}, 80, /*backlog=*/8);
// … 对端 SYN 由测试注入 …
Connection* conn = ln.Accept();  // ESTABLISHED 后非空
```

### `tcp::Connection`

```cpp
// 主动打开
Connection client(&stack);
client.Connect(nic_id, {10,0,0,2}, 50000, {10,0,0,1}, 80);

// 数据
client.Write(data);
auto bytes = conn->Read();
conn->Close();
```

### Stack / Demuxer

- `RegisterTransportEndpoint` — 监听键（UDP Bind / TCP Listen）
- `RegisterConnectedEndpoint` — TCP 四元组
- `UnregisterConnectedEndpoint` — 连接关闭时拆除

`DeliverPacket`（TCP）：**先查四元组，再查监听键**。

## RFC 793 新增路径

| 路径 | 测试 |
|------|------|
| CLOSED → SYN-SENT → ESTABLISHED | `m2_tcp_connect` |
| LISTEN 保持 + 子 Connection | `m2_tcp_handshake` / `m2_tcp_transfer` |
| backlog 满丢弃 SYN | `m2_tcp_backlog` |

详见 [tcp-rfc793-states.md](tcp-rfc793-states.md)。
